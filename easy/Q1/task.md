# User Login – Object Literal & DOM Manipulation

##  Project Description
This project demonstrates **JavaScript Object Fundamentals** using a simple
real-world **User Login** scenario.

The application allows a user to:
- Enter name and email
- Validate inputs using JavaScript
- Login successfully with a popup message
- Update login status dynamically using DOM manipulation

---

## Concepts Covered
- Object Literal
- Object Methods
- `this` keyword
- DOM Manipulation
- Event Handling
- Input Validation using Regex
- Basic UI styling with CSS

---


---

##  Input Validation Rules
- **Name** → Only characters (A–Z, a–z) and spaces allowed
- **Email** → Only characters with `@` and `.` symbols (no numbers)

---

##  How It Works
1. User enters name and email
2. JavaScript validates the input
3. Object method `login()` is called
4. Popup appears: *User logged in successfully*
5. Login status updates on the UI
